Tutorial para crear una clase en java

Utilizando el programa de Netbeans se creara un programa para calcular el sueldo liquido de un empleado si este ha trabajado algunas horas extra, se implementara una clase para quear un objeto que pueda devolver el sueldo liquido en base al salario y horas extra del empleado. Para hacerlo se debera hacer lo siguiente:


  1. Se creara un nuevo proyecto en java en el cual se creara una clase predeterminada llamada "Horasextra", dentro de la clase se declarara el atributo privado llamado "sueldoliquido" para proteger esta variable. Tambien se imporatara la libreria Scanner que se utilizara despues

  2. luego se declarara un metodo publico de tipo de dato real para obtener por medio de un "get" el calculo del sueldo liquido, los argumentos necesarios son el salario y el numero de horas extra trabajadas. El valor que va a retornar sera el 1% del salario por las horas extras trabajadas y este dato se le sumara al salario ordinario.

  3. Con la clase ya creada se procedera a leer dentro del metodo main el salario y numero de horas extra que ingresa el usuario.

  4. Luego se creara una nueva variable del tipo de la clase que se creo donde se guardara el resultado.

  5. Finalmente se imprimira en pantalla el resultado con la nueva variable seguida de punto y obtendra las variables solicitadas para devolver el calculo.

  6. Este es un ejemplo acerca de la programacion orientada a objetos.
